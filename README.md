# Customer Data Cleaning Project

## Overview
This project demonstrates data preprocessing and quality checking using Python Pandas on a customer dataset.

## Tasks Performed
- Loaded the customer dataset
- Checked for missing values
- Checked for duplicate records
- Standardized column names
- Generated descriptive statistics
- Exported the processed dataset

## Technologies Used
- Python
- Pandas
- VS Code

## Output
Generated a processed customer dataset ready for analysis.
